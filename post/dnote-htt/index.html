1.1 a dependent type over a type

$P$ be a dependent type over $A$, $Q$ be a dependent type over $P$
$Q(a,u)$ is a type for every $a:A$ and $u:P(a)$

a dependent type is a map of a type into a universe

1.1.1 Dependent products
terms of $\prod (a:A), P(a)$ :=
sections of $P$ :=
functions from a term of $A$ to a term of type $P(a)$.

each judgement defines a term:
$a:A \vdash g(a):P(a)$ defines $\lambda a.g(a):\prod(a:A),P(a)$

$g(a):P(a)$, $x:A$, $\beta$-rule:
$evaluate(\lambda a.g(a),x)=g(x)$
$f:\prod(a:A),P(a)$, $P(a)$ be a section of $P$, $\eta$-rule
$\lambda a.evaluate(f,a) = f$

$Q$ be a dependent type over a dependent type $P$ over $A$.
$\prod (a:A)(u:P(a)),Q(a,u) := \prod (a:A), (\prod(u:P(a)),Q(a,u))$

1.1.2 Dependent sums
$\langle -, -\rangle : \prod (a:A), (P(a) \to \sum (x:A),P(x))$
includes every $a:A$ and $u:P(a)$ as the pair $\langle a,u \rangle$ in $\sum (a:A),P(a)$.

-- a dependent type over $\sum(a:A),P(a)$,
$Q:(\sum(a:A),P(a)) \to Type$
-- a dependent function maps each pair $a:A$ and $u:P(a)$ a term of type $Q(\langle a, u \rangle)$,
$q:\prod(a:A)(u:P(a)),Q(\langle a,u \rangle)$
-- a section of $Q$,
$\sigma (Q,q):\prod (w:\sum(a:A),P(a)),Q(w)$
$\sigma (\langle a,u \rangle) = q(a,u)$
