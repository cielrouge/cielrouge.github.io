高效可视化算法两个准则:

2. 动态变化中的拓扑保持

对参数1-流形的 曲率适应、拓扑保持 的逼近.

逐段线性(PL)逼近:
1. 拓扑等价于原来的流形
2. 最小化线性逼近的数量
3. 接受用户指定的距离与曲率的误差边界

为动态分子可视化选择的拓扑等价来自ambient isotopy，保持了时间变化中几何模型的嵌入.

Definition 1.
$X$, $Y$是$R^n$的子集. ambient isotopy 是一个连续函数 $H: R^n \times [0,1] \to R^n$ s.t.
1. $H(\cdot,0)$ 是identity函数
2. $H(X,1)=Y$
3. $\forall t \in [0,1]$, $H(\cdot, t)$ 是$R^3$上同胚。

参数1-流形的曲率和拓扑

记号: $c: [0,1]\to R^3$是$C^3$曲线，记
$$\mu_c([a,b]) = \int_a^b \parallel c”(t)\parallel dt$$

Theorem 1.
$c: [0,1]\to R^3$是$C^3$曲线. 对each $\epsilon > 0$, $\exists n \in \mathbb{N}$, $\exists$ partition $X = \lbrace p_1, …, p_n \rbrace \subset [0,1]$ s.t. $p_1=0,p_n=1,p_1<p_2<…< p_{n-1} < p_n$, and $$\int_{p_i}^{p_{i+1}} \parallel c”(z) \parallel dz = \mu_c ([p_i,p_{i+1}])<\epsilon$$ 有紧致圆柱$\lbrace C_i \rbrace_{i=1}^n$ s.t. 每个$C_i$的轴都与$c(p_i)$处切线对齐, 且有半径$\epsilon$. 依次连接顶点$\lbrace c(p_i): i = 1, …,n \rbrace$形成的折线和曲线$c$都在$\bigcup_{i=1}^n C_i$中.

Proof.
对集合$X$中每个点构造一个圆柱$C_i$. $L_i$是包含向量$c’(p_i)$的线. 考虑以 $c’(p_{i+1})$ 为法向量的平面. 这个平面与$L_i$交于一点，记为$q$. 定义$C_i$为半径为$\epsilon$, 轴是$L_i$中连接$p_i$与$q$的部分, 的圆柱. 由泰勒定理，若$t\in [p_i,p_{i+1}]$, 则 $$\vert c(t) - c(p_i) - (t - p_i) c’ (p_i) \vert \leq \int_{p_i}^t \parallel c”(z) \parallel (t - z) dz \leq \int_{p_i}^{p_{i+1}} \parallel c”(z) \parallel dz < \epsilon$$ $c(t)$距离$c(p_i)+(t-p_i)c’(p_i)$至多$\epsilon$. 故$c(t)$距离$L_i$至多$\epsilon$. 所以$c(t)$在圆柱$C_i$中. 每个圆柱都是凸的, 逼近线段的端点$c(p_i),c(p_{i+1})$在圆柱$C_i$中.

(定理一可用于创造曲线的PL逼近，但造出的逼近不一定与原曲线拓扑等价.需要在$\epsilon$的选择上附加更多限制，使生成的逼近与原曲线环境同痕.)

Corollary 1.
记定理一证明中构造的圆柱的数量为$N(\epsilon)$. 若$c$也是$C^3$的, s.t. $\parallel c”(t) \parallel > 0 \space (\forall t)$, 则 $$ \lim_{\epsilon \to 0} \sqrt{\epsilon} \cdot N(\epsilon) = \int_0^1 \sqrt{\parallel c”(u) \parallel } du$$.

Lemma 1.
$c: [0,1]\to R^3$是$C^3$曲线, $r,s\in [p_i,p_{i+1}]$, 则 $\parallel c’ ( r ) - c’ (s) \parallel \leq \sqrt{3} \epsilon$.

Corollary 2.
$c: [0,1]\to R^3$是$C^3$曲线, $\epsilon < \sqrt{2 / 3}$. 则对点$r,s\in [p_i, p_{i+1}]$, $c’( r )$ 与 $c’(s)$ 之间的切向差不超过$\pi/2$.

Lemma 3.
选择$\gamma$, s.t. $\gamma > 0$, $\gamma << (\kappa / 2)(t - p_i)^2$. for $t \in (p_i, p_{i+1}]$, $S_t$是闭球, 球心: $c(p_i)+(t-p_i) c’(p_i)$, 半径: $r_t=(\kappa / 2)(t - p_i)^2 + \gamma$. 则 $c(t)\in int(S_t)$.
Proof.
by Taylor’s theorem:
$$\begin{align} \parallel c(t) - c(p_i) - (t - p_i)c’(p_i) \parallel & \leq \int_{p_i}^t \parallel c”(z) \parallel (t - z) dz \\
& \leq \kappa [t^2 - (1 / 2)t^2 - (t p_i - (1 / 2) p_i^2)] \\
& = \kappa (1 / 2)(t - p_i)^2 \end{align}$$

for each $t \in (p_i,p_{i+1}]$, 定义”雪锥” $K_t$, 是$S_t$与${p_i}$的凸包.

构造环境同痕

算法梗概:
几何目标: 不断减小用于定义 $V_{i,i+1}$ 的每个圆柱 $C_i$ 的半径, 直至这些 $V_{i,i+1}$ 的内部两两不交.
算法开始: 用 $\delta$ 的seed值来构造集合 $X_1$ s.t. $\mu_c ([p_i,p_{i+1}]) < \delta$ (for each i = 1..n and $p_i, p_{i+1} \in X_1$).
算法进行: 将 $\delta$ 替换为 $\delta / 2$, 直到 bounding volumes 两两不交 (为形成整体曲线的环境同痕).
停止准则: 最小分隔距离

先构造 $c_i$ 上的局部同态，再构造全 $c$ 上的环境同痕.

记 $N_w$ 为 $c_i$ 在 $w$ 处的法平面 (for each $w \in c_i$).
根据推论2确保$c_i$上的切线偏离小于$\pi / 2$.
定义 $h_i : c_i \to a_{i,i+1}$ by $h_i(w)$ 是 $N_w \cap a_{i,i+1}$ 上的单点.

Theorem 2.
$h_i$ 是固定了 $c(p_i)$ 和 $c(p_{i+1})$ 的同态.
Proof.
假设 $a_{i,i+1}$ 是 $N_w$ 的子集，$a_{i,i+1}$ 的两个端点都在 $N_w$ 中.
但这两个端点也是 $c_i$ 中的点，与引理2矛盾.
若 $w$ 是 $c_i$ 端点之一，则 $w$ 也是 $a_{i,i+1}$ 中的点.
但由引理2，这个端点是 $N_w \cap a_{i,i+1}$ 中唯一的元.
故假定 $w$ 不是端点，则由引理2，$c(p_i)$ 也不在 $N_w$ 中. 同理 $c(p_{i+1})$ 也不在 $N_w$ 中.
但因为 $\pi / 2$ 是 $c_i$ 上切向的上界，$c(p_i)$ 和 $c(p_{i+1})$ 必在 $N_w$ 的异侧.
所以 $a_{i,i+1}$ 与 $N_w$ 必定相交.
$a_{i,i+1}$ 与 $N_w$ 交于单独的一点.
于是，$h_i$ 是每段 $c_i$ 上的良定函数，在每段 $c_i$ 上连续.
$h_i$ 保持 $c_i$ 端点固定.
$h_i$ 是满的: $a_{i,i+1}$ 的端点保持固定，$h_i$ 的像是 $a_{i,i+1}$ 的连通子集.
$h_i$ 是1-1的: $a_{i,i+1}$ 在$c_i$ 的 $\epsilon$ 距离内.
$\epsilon < 1 / (2 \kappa)$. $\kappa$ 是最大曲率.
所以，线段 $[w_1,h(w_1)]$ 和 $[w_2,h(w_2)]$ 不交.

在 bounding volumn $V_{i,i+1}$ 中定义局部同痕.
