Hibert’s Nullstellensatz:
(几何对象)仿射簇 $\leftrightarrow$ (代数对象)一个多项式环的根理想

坐标环:
被assign到一个仿射簇的仿射代数

代数 有限生成 iff
同构于$R[x_1,\cdots ,x_n]/I$.

仿射K_代数:
域K上的有限生成代数

(A是域K上的一个代数)
A是整环，且在域K上algebraic => A是域.
A是域，且在一个仿射K-整环中 => A is algebraic.

$A\cong K[x]/I$
$A$是整环 => $I$是素理想, $A$在$K$上algebraic => $I$非零.
$K[x]$是主理想整环, $I=(f)$ ($f\in K[x]$不可约), $I$是最大理想.


(A->B是域K上的代数同态)
若B 有限生成,
则B中最大理想在A中的 原像也是最大理想.

($K$是域, $P=(\xi_1,…,\xi_n)\in K^n$)
$\mathfrak{m}_P:=(x_1 -\xi_1,…,x_n-\xi_n)$
是$K[x_1,…,x_n]$的一个最大理想.

$\forall f\in K[x_1,…,x_n]$ 同余$f(\xi_1,…,\xi_n)$ modulo $\mathfrak{m}_P$ =>
$\mathfrak{m}_P$是$\phi:K[x_1,…,x_n]\to K, f\mapsto f(\xi_1,…,\xi_n)$的核 =>
$K[x_1,…,x_n]/\mathfrak{m}_P\cong K$.


(一个代数闭域上的 一个多项式环的 所有最大理想:)
若$\mathfrak{m}\subseteq K[x_1,…,x_n]$是代数闭域$K$上一个多项式环的一个最大理想，
则存在点$P=(\xi_1,…,\xi_n)\in K^n$, s.t.
$\mathfrak{m}=(x_1 -\xi_1,…,x_n-\xi_n)$

for each $i=1..n, K[x_i]\cap \mathfrak{m}$是$K[x_i]$的一个最大理想.
$K[x_i]$是主理想整环 =>
$K[x_i]\cap \mathfrak{m}$有形式: $(p_i)_{K[x_i]}$
($p_i$是不可约多项式)
(set of all $K[x_i]$-linear combinations of $p_i$)
$K$代数闭 =>
$(p_i)_{K[x_i]}=(x_i-\xi_i)_{K[x_i]}$, $\xi_i\in K$.
$\exists \xi_1,…,\xi_n\in K$, with $x_i-\xi_i\in\mathfrak{m}$
$\mathfrak{m}_P\subseteq \mathfrak{m}$, $\mathfrak{m}_P$是最大理想 =>
$\mathfrak{m}=\mathfrak{m}_P$


$S\subseteq K[x_1,...,x_n]$是多项式的集合. $S$给出仿射簇:
$\mathcal{V}=\mathcal{V}_{K^n}(S)$:=$\lbrace (\xi_1,…,\xi_n)\in K^n\mid f(\xi_1,…,\xi_n)=0,\forall f\in S\rbrace $

(多项式集合给出的仿射簇的)点 $\leftrightarrow$ (含此多项式集合,多项式环的)最大理想:
($K$是代数闭域, $S\subseteq K[x_1,…,x_n]$是多项式的集合)
$\mathcal{M}_S$是满足: $S\subseteq\mathfrak{m}\subset K[x_1,…,x_n]$
的所有最大理想 的集合.
则 $\Phi:\mathcal{V}\to \mathcal{M}_S, (\xi_1,…,\xi_n)\mapsto (x_1-\xi_1,…,x_n-\xi_n)$ 是双射.

Hilbert’s Nullstellensatz (1)
($K$是代数闭域, $I$是多项式环的真理想)
$\mathcal{V}(I)\neq \emptyset$

若不存在$g_1,...,g_n\in K[x_1,...,x_n]$ s.t.
$\sum_{i=1}^m g_i f_i =1$,
且K代数闭, 则以下方程组有解:
$f_i (\xi_1,…,\xi_n)=0$ for $i=1..m$.
