($K$:交换环, $A$:$K$-代数 $M$:$A$-模)
$M$ is Noetherian:
$\exists M_1\subseteq M_2\subseteq … \subseteq M$ $\quad$ =>
$\exists n_0\in \mathbb{N}$ s.t. $M_n=M_{n_0}, \forall n \geq n_0$
$M$ is artinian:
$\exists M\supseteq M_1 \supseteq M_2 \supseteq …$ $\quad$ =>
$\exists n_0\in \mathbb{N}$ s.t. $M_n=M_{n_0}, \forall n \geq n_0$

若$A$是域$K$上代数, 则任意有限维$A$-模作为$K$-向量空间, is Noetherian and artinian.

($A$:环, $M$: $A$上左模)
$M$ is Noetherian <=>
$M$的所有子模都是有限生成的 <=>
对$M$的任意子模$N$, $N$与$M/N$ are Noetherian
